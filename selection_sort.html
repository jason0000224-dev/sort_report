選擇排序演算法解說

C 語言實作

1void selection_sort(int arr[], int n) {
2    int i, j, min_idx, temp;
3    for (i = 0; i < n - 1; i++) {       // 每個回合
4        min_idx = i;
5        for (j = i + 1; j < n; j++) {     // 尋找最小值
6            if (arr[j] < arr[min_idx]) {   // 比較
7                min_idx = j;               // 更新最小值位置
8            }
9        }
10        // 交換最小值
11        temp = arr[i];
12        arr[i] = arr[min_idx];
13        arr[min_idx] = temp;
14    }
15}

複雜度分析

時間複雜度
  最佳 O(n²)
  平均 O(n²)
  最差 O(n²)

空間複雜度
  O(1)         ← 原地排序

穩定性
  不穩定（Unstable）

特性
  簡單直觀、交換次數較少
  總是 O(n²)，無法優化

核心概念

每一回合從未排序部分中「尋找」最小的元素，將其與當前位置交換 → 最小值「被選中」並就定位。
重複 (n-1) 回合，每回合確保當前輪未排序部分的最小值就定位，前面已排序的部分不再改變。
