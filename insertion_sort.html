插入排序演算法解說

C 語言實作

1void insertion_sort(int arr[], int n) {
2    int i, j, key;
3    for (i = 1; i < n; i++) {         // 從第二個元素開始
4        key = arr[i];                 // 取出當前元素
5        j = i - 1;
6        while (j >= 0 && arr[j] > key) { // 向後掃描
7            arr[j + 1] = arr[j];     // 向右位移
8            j--;                      // 繼續向左檢查
9        }
10        arr[j + 1] = key;             // 將 key 插入正確位置
11    }
12}

複雜度分析

時間複雜度
  最佳 O(n)    ← 已排序時
  平均 O(n²)
  最差 O(n²)

空間複雜度
  O(1)         ← 原地排序

穩定性
  穩定（Stable）

特性
  簡單直觀、在線可用
  適合小規模與近乎有序資料

核心概念

從第二個元素開始，逐次「取出」元素，與已排序部分從右到左比較。
當元素 > 已排序元素時，該已排序元素「位移」右移一格，直到找到正確位置後「插入」。
最終未排序部分全部被取出並插入至已排序部分中，完成排序。
