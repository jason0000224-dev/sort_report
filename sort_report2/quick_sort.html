快速排序演算法解說

C 語言實作

1int partition(int a[], int low, int high) {
2  int pivot = a[high];
3  int i = low;
4  for (int j = low; j < high; j++) {
5    if (a[j] < pivot) { swap(&a[i++], &a[j]); }
6  }
7  swap(&a[i], &a[high]);
8  return i; /* pivot 最終位置 */
9}
10void quicksort(int a[], int low, int high) {
11  if (low < high) {
12    int p = partition(a, low, high);
13    quicksort(a, low, p - 1);
14    quicksort(a, p + 1, high);
15  }
16}

複雜度分析

時間複雜度
  最佳 O(n log n)
  平均 O(n log n)
  最差 O(n^2)   ← 已排序或極端切分

空間複雜度
  O(log n) (遞迴堆疊)

穩定性
  不穩定（Unstable）

特性
  分治法、高效率於大多數情況
  實作中基準點選擇很重要

核心概念

選取一個基準值(pivot)，將陣列分成小於與大於 pivot 的兩部分，遞迴排序兩邊；pivot 最終就位。
