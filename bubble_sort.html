氣泡排序演算法解說

Python 實作

def bubble_sort(arr):
    n = len(arr)
    for i in range(n - 1):           # 回合
        for j in range(n - 1 - i):   # 比較
            if arr[j] > arr[j + 1]:  # 需交換？
                arr[j], arr[j+1] = \
                  arr[j+1], arr[j]   # 交換
    return arr

複雜度分析

時間複雜度
  最佳 O(n)    ← 已排序時
  平均 O(n²)
  最差 O(n²)

空間複雜度
  O(1)         ← 原地排序

穩定性
  穩定（Stable）

特性
  簡單易懂、教學常用
  大資料效能較差

核心概念

每一回合從左到右掃描，遇到「前大後小」就交換 → 最大值像「氣泡」一樣浮到最右側。
重複 (n-1) 回合，每回合確保當前最大值就定位，後面已排序的部分不再比較。
